# 🚀 Изучение JavaScript — от Hello, world! до реального приложения

Этот проект — серия простых, но полных уроков по JavaScript. Каждый шаг добавляет новые концепции и сразу показывает их в действии.

## 📚 Содержание

| Шаг | Тема | Файл | Что научитесь делать |
|-----|------|------|----------------------|
| 1 | Hello, world! | — | `console.log()`, основы |
| 2 | Калькулятор + условия | `step2.html` | `input`, `if/else`, умножение |
| 3 | Циклы и списки | `step3.html` | `for`, массивы, добавление/удаление |
| 4 | Функции и объекты | `step4.html` | `function`, объекты-словари |
| 5 | filter, map, sort | `step5.html` | обработка массивов данных |
| 6 | Асинхронность (fetch) | `step6.html` | запросы к API, `async/await` |
| 7 | Состояние + рендеринг | `step7.html` | реактивный UI, корзина товаров |
| 8 | Роутинг (SPA) | `step8.html` | переключение страниц без перезагрузки |
| 9 | Формы и валидация | `step9.html` | проверка email, пароля, регистрация |
| 10 | Даты и таймеры | `step10.html` | `setInterval`, `setTimeout`, часы |
| 11 | localStorage | `step11.html` | сохранение заметок, темы |
| Пример | Мини-сайт | `index.html` | всё вместе: заметки + таймер + тема |

## 🎯 Быстрый старт

1. Скачайте любой `.html` файл из уроков
2. Откройте его в браузере (двойным кликом)
3. Изучайте код — он полностью рабочий и самодостаточный
4. Открывайте консоль (F12) — там много полезных выводов
